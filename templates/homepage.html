2022/2023 Advanced Programming project of Veronica Boccaletti, Viola Cimatti, Giulia Tavacca, Anna Ballestrazzi


This software has been developed with the purpose of giving some information about the human genome, taking as input the data contained in a GFF3 file format.
The software reads and analyses the data by processing the information in different active operations.
The user is dealing with a web-based interface. This is the homepage of the application, the interactive buttons provide the access to the other available views.

Active operations

Click the button to see a list of the available operations. Then select an active operation to process the dataset, the result of the execution will be visible in a further view.

Project documentation

Click the button to read a more in depth description of the project. CRC cards and UML class diagrams describe the software analysis and the design, respectively.
